ONKOSULLAR
───────────────────────────────

  NOT: PostgreSQL gerekmez! Veritabani olarak SQLite
  kullanilmaktadir (otomatik olusturulur, ek kurulum yok).

  Kontrol icin: ONKOSULLER_KONTROL.bat cift tiklayin.

3. BASLAT.bat cift tiklayin
   Tarayici otomatik acilacak: http://localhost:3000

SORUN GIDERME
───────────────────────────────
- "Port kullanimda": BASLAT.bat yerine terminalde calistirin:
  npm run dev -- -p 3001

- Veritabani sifirlama:
  npx prisma db push --force-reset
  npm run db:seed

- Veritabani dosyasi: prisma\ecosfer.db
  (Silip KURULUM.bat tekrar calistirarak sifirlanabilir)
